FTK Imager is a lightweight digital forensic tool used to create exact copies of digital storage devices. Investigators use it to produce bit-for-bit forensic images of hard drives, USB devices, and other storage media. The tool also allows examiners to safely preview data without modifying the original device. In addition, FTK Imager can generate hash values such as MD5 and SHA1, which are used to verify that the copied image is identical to the original evidence.

Autopsy is a digital forensics investigation platform built on The Sleuth Kit. It is used by investigators to analyze disk images, recover deleted files, examine system and web artifacts, and generate forensic reports. The tool provides a graphical interface that helps examiners perform complete digital investigations from evidence ingestion to reporting.

Download the Autopsy PDF Document

Wireshark is a free, open-source network protocol analyzer used to capture and analyze network traffic in real time. It is commonly used by cybersecurity professionals, network administrators, and students to understand how data moves across a network and to troubleshoot network problems. Wireshark can capture packets from different network interfaces and display detailed information about each packet, including protocol layers such as TCP, HTTP, DNS, and TLS. The software includes powerful filtering tools that allow users to isolate specific traffic by IP address, protocol, port, or MAC address. This makes it easier to detect suspicious activity, investigate network issues, and analyze communication patterns."
